### –¶–∏–∫–ª—ã while –∏ do-while

–ú–Ω–æ–≥–æ –æ –Ω–∏—Ö –≥–æ–≤–æ—Ä–∏—Ç—å –Ω–µ –±—É–¥–µ–º(–æ–Ω–∏ –æ—á–µ–Ω—å –ø–æ–Ω—è—Ç–Ω—ã –ø–æ—Å–ª–µ for), –ø—Ä–æ—Å—Ç–æ —Ä–∞—Å—Å–º–æ—Ç—Ä–∏–º –ø—Ä–∏–º–µ—Ä—ã —Å–∏–Ω—Ç–∞–∫—Å–∏—Å–∞. –ó–∞ –¥–æ–ø –ø–æ—è—Å–Ω–µ–Ω–∏—è–º–∏ –¥–æ—Ä–æ–≥—É –º—ã –ø–æ–º–Ω–∏–º)

–ü—Ä–∏–º–µ—Ä do-while:

### –û–ø–µ—Ä–∞—Ç–æ—Ä—ã break –∏ continue

–í —è–∑—ã–∫–µ C++ —Å—É—â–µ—Å—Ç–≤—É—é—Ç –æ–ø–µ—Ä–∞—Ç–æ—Ä—ã `break` –∏ continue, –∫–æ—Ç–æ—Ä—ã–µ –∏—Å–ø–æ–ª—å–∑—É—é—Ç—Å—è –¥–ª—è —É–ø—Ä–∞–≤–ª–µ–Ω–∏—è —Ö–æ–¥–æ–º –≤—ã–ø–æ–ª–Ω–µ–Ω–∏—è —Ü–∏–∫–ª–∏—á–µ—Å–∫–æ–≥–æ –ø—Ä–æ—Ü–µ—Å—Å–∞. –≠—Ç–∏ –æ–ø–µ—Ä–∞—Ç–æ—Ä—ã –º–æ–≥—É—Ç –±—ã—Ç—å –ø—Ä–∏–º–µ–Ω–µ–Ω—ã —Ç–æ–ª—å–∫–æ –≤ —Ç–µ–ª–µ —Ü–∏–∫–ª–∞.

–û–ø–µ—Ä–∞—Ç–æ—Ä break –ø—Ä–µ–¥–Ω–∞–∑–Ω–∞—á–µ–Ω –¥–ª—è –∏—Å–∫—É—Å—Å—Ç–≤–µ–Ω–Ω–æ–≥–æ –ø—Ä–µ—Ä—ã–≤–∞–Ω–∏—è –≤—ã–ø–æ–ª–Ω–µ–Ω–∏—è:

–ß–∞—â–µ –≤—Å–µ–≥–æ –æ–ø–µ—Ä–∞—Ç–æ—Ä break –∏—Å–ø–æ–ª—å–∑—É–µ—Ç—Å—è –≤ —Å–æ—á–µ—Ç–∞–Ω–∏–∏ —Å –æ–ø–µ—Ä–∞—Ç–æ—Ä–æ–º —É—Å–ª–æ–≤–∏—è if. –í —ç—Ç–æ–º —Å–ª—É—á–∞–µ, –ø—Ä–æ–∏—Å—Ö–æ–¥–∏—Ç –ø—Ä–æ–≤–µ—Ä–∫–∞ –Ω–µ–∫–æ—Ç–æ—Ä–æ–≥–æ —É—Å–ª–æ–≤–∏—è, –∏ –≤ —Å–ª—É—á–∞–µ —É—Å–ø–µ—Ö–∞ –≤—ã–∑—ã–≤–∞–µ—Ç—Å—è –æ–ø–µ—Ä–∞—Ç–æ—Ä break.
